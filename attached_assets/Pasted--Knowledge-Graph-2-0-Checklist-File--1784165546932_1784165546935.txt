## معيار الإغلاق النهائي

يُعتبر Knowledge Graph 2.0 جاهزًا عندما:

* لا توجد Node بلا provenance أو confidence.
* لا توجد Edge بلا relationType أو evidence.
* يمكن تمييز structural / behavioral / runtime / heuristic بوضوح.
* الاستعلامات تدعم الفلترة حسب الثقة والمصدر والدلالة.
* الـ dashboard يعرض الأدلة والثقة وليس الروابط فقط.
