**الهدف:** التأكد أن كل endpoint له سلسلة كاملة: spec → route → handler → DB → audit → frontend → generated client → tests.

**الملفات:**

* `lib/api-client-react/src/generated/*`

* `artifacts/api-server/src/routes/*.test.ts`

**Checklist:**

* [ ] مطابقة كل path في `openapi.yaml` مع route فعلي بنفس السلوك.
* [ ] التأكد أن `api-zod` و`api-client-react` ناتجان من نفس spec الحالية.
* [ ] حصر أي endpoint لا يملك test مباشر أو سلوك error واضح.

  `projects.ts`, `discovery.ts`, `tasks.ts`, `workflows.ts`, `ai.ts`, `graph.ts`.

**Done criteria:**

* لا يوجد endpoint يتيم.
* لا يوجد mismatch بين spec والتنفيذ والـ generated artifacts.
* كل فجوة موثقة بسبب واضح أو مُغلقة بكود/اختبار.
